== Description ==

Le plugin WP Social Icon vous permet de placer sur votre site des icones de vos réseaux sociaux préférés.

WP Social Icons plugin allows you to place on your website icons of your favorite social networks.

Langues disponibles / Languages :

* Français
* English

Fichier wp-socialicon.pot disponible / wp-socialicon.pot file available

== Installation ==

= FR =

1. Ajouter le répertoire 'wp-socialicon' dans le répertoire '/wp-content/plugins/'
2. Activer le plugin via le menu 'Plugins' dans WordPress
3. Aller dans 'Réglages->WP SocialIcon' dans l'interface d'administration pour sélectionner vos options
4. Placer ce shortcode à l'endroit que vous souhaitez : [wpsocialicon]

= EN =

1. Upload the full directory into your '/wp-content/plugins' directory
2. Activate the plugin at the plugin administration page
3. Open the plugin configuration page, which is located under 'Option->WP SocialIcon'
4. Place this shortcode to the place you want : [wpsocialicon]
